# UART_Read Function

## C

```c
uint32_t UART_Read(uint8_t *rb, const uint32_t len, const int32_t wait);
```

## Description

 This function reads the SERCOM UART for the given length.

## Parameters

| Param | Description |
|:----- |:----------- |
| rb | read buffer |
| len | length to read |
| wait | wait timeout  

## Returns

 None
