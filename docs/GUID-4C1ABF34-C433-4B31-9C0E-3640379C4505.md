# UART\_ERROR Enum

**Parent topic:**[DFU API Usage](GUID-601EC36F-434F-4CEE-BF96-232B23F39458.md)

## C

```c
typedef enum
{
    // UART - No Error
    UART_ERROR_NONE = 0,
    // UART - Overrun error
    UART_ERROR_OVERRUN = 0x02,
    // UART - Framing error
    UART_ERROR_FRAMING = 0x04,
    // UART - Parity error
    UART_ERROR_PARITY = 0x08
    
} UART_ERROR;

```
