# PDS\_Operation\_Offset\_t Struct

**Parent topic:**[PDS Library Usage](GUID-A7B0958C-E476-48EA-9C30-DA83508CC577.md)

## C

```c
typedef struct
{
    // Item identifier
    PDS_MemId_t id;
    // Item offset
    uint16_t    offset;
    // size of item
    uint16_t    size;
    // corresponding ram address for the item
    uint8_t     *ramAddr;
}PDS_Operation_Offset_t;

```
