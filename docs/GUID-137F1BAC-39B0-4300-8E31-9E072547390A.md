# Crc32Add Function

## C

```c
void Crc32Add(uint8_t* pBuff, uint32_t bSize);
```

## Description

 This routine adds the given data into CRC32 caculation and calculates CRC

## Parameters

| Param | Description |
|:----- |:----------- |
| uint8_t pBuff | pointer to the data to add |
| uint32_t bSize | size of the data to add  

## Returns

 None
