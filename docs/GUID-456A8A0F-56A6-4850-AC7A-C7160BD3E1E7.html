DEVICE_EnterDeepSleep Function

C

void DEVICE_EnterDeepSleep(bool enableRetRam,  uint32_t interval);

Description

The API is used to enter deep sleep mode.

Parameters

ParamDescription
enableRetRamEnable/Disable retention ram.

Parameters

ParamDescription
intervalThe interval of deep sleep mode (unit: ms). Set as 0 will keep the device in the deep sleep mode until the INT0 (PB9) is triggered.

Parameters

None

Returns

None
